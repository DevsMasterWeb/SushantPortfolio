# **App Name**: PersonaLink

## Core Features:

- Profile Display: Display a clean and professional profile with a photo, bio, and key skills. This feature should allow the user to present an introduction of themselves and highlight the critical points for the viewer.
- Project Showcase: Showcase selected projects with descriptions, images, and links to live sites or repositories.
- Skills Visualization: Visually represent skills using a dynamic and engaging format. Incorporate animations or interactive elements.
- Contact Form: Enable visitors to send messages directly through the website.
- Resume Integration: Link to or embed a resume for easy access by potential employers.
- AI-Powered Personalization Tool: Implement a generative AI tool to provide personalized recommendations based on the content the portfolio user inputs (projects, skills). The personalization tool is capable of using reasoning to enhance the display of the most relevant content on the site based on its assessment of the user viewing it.

## Style Guidelines:

- Primary color: Slate blue (#5F9EA0) to convey professionalism and reliability.
- Background color: Very light grey (#F0F8FF) for a clean and modern feel.
- Accent color: Soft beige (#F5F5DC) to add warmth and highlight key sections.
- Body font: 'Inter', sans-serif, for clear, readable text.
- Headline font: 'Space Grotesk', sans-serif, for impactful headlines.
- Use simple, outline-style icons to represent skills and sections.
- A clean, single-page layout with clear sections and smooth scrolling.
- Subtle hover effects and transitions to add interactivity without overwhelming the user.
